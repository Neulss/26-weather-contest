# 2026 날씨 축산 데이터 분석

## 프로젝트 개요
축산 데이터와 기상 데이터를 통합하여 분석하는 프로젝트입니다.

## 폴더 구조
```
2026_날씨_축산데이터/
├── 축산 EDA_0508.ipynb  # EDA 분석 노트북
└── hanwoo/              # 축산 관련 데이터
    ├── hanwoo_train.csv
    ├── hanwoo_weather.csv
    ├── hanwoo_area.csv
    ├── hanwoo_death.csv
    ├── hanwoo_lineage.csv
    ├── META_ASOS.csv
    ├── META_AWS.csv
    ├── asos_new.csv
    └── aws_new.csv
```

## 사용 데이터
- 축산 데이터: 한우 관련 통계
- 기상 데이터: ASOS, AWS 관측소 데이터

## 분석 내용
- 기상 데이터와 축산 데이터의 상관관계 분석
- 지역별 기상 패턴 분석
- 시계열 데이터 처리 및 시각화
